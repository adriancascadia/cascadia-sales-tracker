# SalesForce Tracker - User Guide

## Getting Started

### Logging In
1. Open the SalesForce Tracker app
2. Click "Login with Manus"
3. Enter your credentials
4. You'll be redirected to the dashboard

### Dashboard Overview
The dashboard shows your key metrics:
- **Total Customers** - Number of customers in your territory
- **Today's Visits** - Visits scheduled for today
- **Total Visits** - All-time visits completed
- **Total Orders** - Orders created
- **Quick Actions** - Fast access to common tasks
- **Live Tracking** - View team locations

## Sales Rep Features

### Managing Customers

**Viewing Customers**
1. Click "Customers" in the sidebar
2. Browse the customer list or search by name
3. Click a customer to view details
4. Click "History" to see interaction timeline

**Adding a Customer**
1. Click "Customers" → "Add Customer"
2. Fill in customer details (name, address, phone)
3. Enter location (latitude/longitude or use map)
4. Click "Save"

**Bulk Import**
1. Click "Customers" → "Import"
2. Download the CSV template
3. Fill in customer data
4. Upload the file
5. Review and confirm import

### Creating Visits

**Start a Visit**
1. Click "Visits" or "Start New Visit" on dashboard
2. Select a customer
3. Click "Check In"
4. App captures GPS location and time
5. Add notes if needed

**During Visit**
- Take photos of products/displays
- Create orders for the customer
- Add internal notes
- View customer history

**End Visit**
1. Click "Check Out"
2. Add visit notes (optional)
3. Click "Confirm"
4. Visit is recorded with duration

### Creating Orders

**Add Order**
1. During a visit, click "Create Order"
2. Select products and quantities
3. Choose distributor
4. Review total
5. Click "Create Order"

**View Orders**
1. Click "Orders" in sidebar
2. See all orders you've created
3. Filter by customer, date, or status
4. Click order to view details

### Uploading Photos

**During Visit**
1. Click "Take Photo"
2. Take photo with device camera
3. Add photo type (before/after/other)
4. Add description (optional)
5. Click "Upload"

**From Photo Gallery**
1. Click "Photo Gallery"
2. Browse all photos
3. Filter by rep, type, or date
4. Click photo to view full size
5. Download if needed

### Tracking Mileage

**Start Mileage Log**
1. Click "Mileage Tracking"
2. Click "Start Mileage Log"
3. Enter start odometer reading
4. Click "Start"

**End Mileage Log**
1. Click "End Mileage Log"
2. Enter end odometer reading
3. Add notes (optional)
4. Click "Save"
5. Distance calculated automatically

**View History**
1. Scroll down to see all mileage logs
2. See distance, duration, date
3. Delete logs if needed

### Route Optimization

**Optimize Your Route**
1. Click "Route Optimization"
2. Select optimization method
3. Set max stops per route
4. Click "Optimize Route"
5. View suggested route sequence
6. See distance and time savings

**Follow Route**
1. View route details
2. Click "Start Visit" for each stop
3. Navigate using GPS
4. Complete visits in order

### Offline Mode

**Working Offline**
- App automatically detects internet loss
- All operations are queued locally
- Changes sync when reconnected
- Offline indicator shows status

**Offline Indicator**
- **Green** - Online and synced
- **Yellow** - Offline (no internet)
- **Blue** - Syncing operations
- **Orange** - Pending changes

**Offline Settings**
1. Click "Offline Settings"
2. View cache status
3. Clear cache if needed
4. See pending operations

## Manager Features

### Viewing Performance

**Performance Analytics**
1. Click "Performance Analytics" in sidebar
2. View interactive charts:
   - Sales trends over time
   - Rep performance rankings
   - Customer growth
   - Order volume by distributor
   - Visit completion rates
   - Photo upload trends
   - Rep activity

**Filtering Charts**
1. Select date range (7/30/90 days, 1 year)
2. Use custom date picker
3. Charts update automatically

### Reports

**Sales Reports**
1. Click "Reports" in sidebar
2. View sales performance by rep
3. See customer visit frequency
4. Check order volume by distributor
5. Review route efficiency metrics
6. View photo documentation

**Export Reports**
1. Click "Export CSV" on any report section
2. File downloads to your device
3. Open in Excel or Google Sheets

### Mileage Reimbursement

**View Mileage Reports**
1. Click "Mileage Reports"
2. View summary statistics
3. Filter by rep, date, or status
4. See reimbursement calculations
5. Export for accounting

**Reimbursement Rate**
- Uses IRS standard rate: $0.655/mile
- Automatically calculated
- Updated annually

### Live Tracking

**View Team Locations**
1. Click "Live Tracking"
2. See team members on map
3. Click rep to view details
4. See current location, last update
5. View active visits

### Managing Alerts

**Create Alert**
1. Click "Alerts"
2. Click "Create Alert"
3. Select type (visit, order, performance)
4. Set conditions
5. Click "Save"

**View Alerts**
1. Click "Alerts"
2. See all active alerts
3. Click to view details
4. Mark as read when addressed

### Team Dashboard

**Manager Dashboard**
1. Click "Manager Dashboard"
2. View team overview
3. See key metrics
4. Monitor active visits
5. Check pending orders
6. Review alerts

## Mobile App

### Installing App

**On iPhone**
1. Open app in Safari
2. Tap Share button
3. Tap "Add to Home Screen"
4. Name the app
5. Tap "Add"

**On Android**
1. Open app in Chrome
2. Tap menu (three dots)
3. Tap "Install app"
4. Confirm installation
5. App appears on home screen

### Using Mobile App
- Works offline automatically
- Faster than browser
- Push notifications for alerts
- App shortcuts for quick actions

## Troubleshooting

### Can't Log In
- Check internet connection
- Verify credentials
- Clear browser cache
- Try different browser
- Contact support

### Photos Not Uploading
- Check file size (max 10MB)
- Verify internet connection
- Try different photo format
- Clear browser cache
- Restart app

### Offline Mode Issues
- Check offline indicator
- Clear offline cache
- Restart app
- Check internet connection
- Review pending operations

### Performance Issues
- Clear browser cache
- Close other tabs
- Restart browser
- Update app
- Contact support

## Keyboard Shortcuts

- **Ctrl/Cmd + N** - New visit
- **Ctrl/Cmd + O** - New order

- **Enter** - Confirm actions
